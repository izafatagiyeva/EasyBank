# EasyBank
## About The Project
This is a modern, fully responsive website built using **HTML5** and **SCSS**. The project focuses on clean architecture, modular style management, and responsive web design principles. Rather than using vanilla CSS, the styling is structured with SCSS to leverage variables, nesting, and mixins for a highly maintainable codebase.

## Features
* **Fully Responsive Layout:** Optimized for seamless viewing on mobile phones, tablets, and desktop screens using custom media queries.
* **Modular SCSS Architecture:** Organized using partials (`_header.scss`, `_variables.scss`, etc.) for easy scalability and maintenance.
* **Dynamic Styling with Variables:** Utilizes SCSS variables for centralized control over colors, typography, and spacing.
* **Clean UI/UX:** Features modern design components with smooth CSS transitions and interactive hover effects.
* **Cross-Browser Compatibility:** Thoroughly tested to ensure pixel-perfect rendering across modern web browsers.
